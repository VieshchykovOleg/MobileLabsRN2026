# GestureClicker — Лабораторна робота №3

## Тема
Використання кастомних жестів у React Native та стилізація інтерфейсу мобільного застосунку.

## Встановлення та запуск

```bash
cd lab3
npm install
npx expo start
```

## Реалізований функціонал

### Екран "Гра" — головний клікер
Центральний об'єкт реагує на 6 типів жестів:

| Жест | Handler | Дія |
|---|---|---|
| 👆 Tap | `TapGestureHandler` | +1 очко |
| ✌️ Double Tap | `TapGestureHandler (numberOfTaps=2)` | +2 очки |
| ⏱️ Long Press | `LongPressGestureHandler (3000ms)` | +5 очок |
| ↔️ Pan | `PanGestureHandler` | Переміщення по екрану |
| 👉👈 Fling | `FlingGestureHandler` | +1–10 випадкових очок |
| 🔍 Pinch | `PinchGestureHandler` | Масштабування, +3 очки |

Також реалізовано анімацію пульсу при кліку (`Animated.sequence`).

### Екран "Завдання"
9 завдань з прогрес-баром та статусом виконання:
- Зробити 10 кліків
- Подвійний клік 5 разів
- Утримати 3 секунди
- Перетягнути об'єкт
- Свайп вправо / вліво
- Змінити розмір (pinch)
- Набрати 100 / 500 очок (власне завдання)

### Екран "Налаштування"
- Перемикач світлої/темної теми
- Статистика (рахунок, виконані завдання)
- Довідка по жестах
- Кнопка скидання прогресу з підтвердженням

### Стилізація
- Динамічна тема через React Context (`ThemeProvider`)
- Підтримка світлої та темної теми
- Стилі через `StyleSheet.create` з об'єктом теми

## Висновки

Під час виконання лабораторної роботи було вивчено роботу з жестовими обробниками бібліотеки `react-native-gesture-handler`. Реалізовано 6 типів жестів: TapGestureHandler (одиночний та подвійний клік), LongPressGestureHandler, PanGestureHandler, FlingGestureHandler та PinchGestureHandler. Особливістю є коректне вкладення обробників — порядок важливий для уникнення конфліктів між жестами. Стилізацію реалізовано через власну систему тем на базі React Context, що забезпечує підтримку світлої та темної теми у всьому застосунку.
